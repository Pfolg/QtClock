一个简单的桌面时钟，基于Qt，使用PySide6编写。


### 功能

显示时间

### 可选设置

+ 更改字体&颜色
+ 更改时钟在屏幕上的位置
+ 使用RGB效果

你还可以更改这及其垃圾的源代码
